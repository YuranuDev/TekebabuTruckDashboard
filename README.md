# TekebabuTruckDashboard
ETS2向け車両コントロールダッシュボード<br>
vJoy仮想コントローラで動作します。

# クレーマーに向けての注意
・このソフトはまじめに作ってないのでバグだらけです<br>
・タッチパネル対応のPCでしか使用できません<br>
→クリックは利きません。マウスでクリックするくらいならキーボードにボタン割り当ててね♪<br>

## ・How to Setup?
1. https://github.com/shauleiz/vJoy/releases よりvJoyをインストール<br>
2. 「Configure vJoy」を起動し、vJoy DeviceID 1を選択し、 ボタン数を32個に増やす<br>
<img width="370" height="490" alt="image" src="https://github.com/user-attachments/assets/32693104-24ac-42f3-a086-472f8b914c2d" /><br>
3. Serverを起動して初期設定<br>
4. Clientを起動してサーバーのIPとポート打つ<br>
5. ETS2でバインド設定(コントローラーの項目でvJoy Deviceを必ず追加すること)

# 6. Fin
100% working lets go!
